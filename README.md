## Code Example

```py
import datetime

name = input(f"Give me your name? : ")
age = int(input(f"Now, Give me your age? : "))

current_year = datetime.datetime.now().year

year = ((current_year - age) + 100)

print(f"{name} will be 100 years old in the year {year}")
```
